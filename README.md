# Obstacle Avoidance Bot
A bot utilizing an ultrasonic sonar sensor mounted on a servo to scan the frontal area and avoid obstacles based on the gathered data.
<br>Materials:<br>
1x L298N motor driver, 4x tt motors, 4x plastic wheels, jumper cables, 1x esp32, 1x sg90 servo, 1x hc-sr04 ultrasonic sonar sensor, 1x foam sheet, 1x Lipo battery, 1x XT-60 connector, 1x lm2596 buck converter
<br>Note:
1. Rename the main [obstacle avoidance bot].py file to main.py and then upload it via Thonny
2. You can change the pinouts of the code as you wish
3. You can tune the servo angles to your liking or setup
4. Make sure to have a great day
