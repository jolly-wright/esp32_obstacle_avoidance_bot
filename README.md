# Obstacle Avoidance Bot
A bot utilizing an ultrasonic sonar sensor mounted on a servo to scan the frontal area and avoid obstacles based on the gathered data.
Materials:
1x L298N motor driver, 4x tt motors, 4x plastic wheels, jumper cables, 1x esp32, 1x sg90 servo, 1x hc-sr04 ultrasonic sonar sensor, 1x foam sheet
<br>Note:
1. You can change the pinouts of the code as you wish
2. You can tune the servo angles to your liking or setup
3. Make sure to have a great day
